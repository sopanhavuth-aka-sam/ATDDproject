Acoustic Telemetry Data Display(ATDD)

Developer Guide

What is it?

The Acustic Telemetry Data Display (ATDD) system is a web based application that takes output data from underwater receivers (provide by Shark Lab of California State University, Long Beach) to create animations that display shark and fish detections along the California coast.

The Latest Version

Details about the latest version can be found at project's github page.

Features

  • Select a species or a specific individual to track.
  • Select a time period for which to track your selection.
  • See specific details about a detection, the latitude, longitude, date and time the individual was detected.
  • Choose how detections are to be displayed per animation frame hourly, daily, weekly or monthly.

Components and Dependencies

  • D3 javascript library
  • JQuery javascript library
  • Mapbox
  • Select2 plugin
  • Zebra_Datepicker plugin
